LRU Cache Manager

Least Recently Used Cache with O(1) Operations

Result:

Perform an operation to see results

Cache State:

Cache updates automatically

How it works:

  • PUT: Insert or update a key-value pair in the cache
  • GET: Retrieve a value by key. Returns -1 if key not found
  • Cache capacity is 5. When full, the least recently used item is removed

LRU Cache Mini Project | Time Complexity: O(1) for all operations
